README

1. Datasets

This benchmark includes four datasets: Freshretailnet (retail demand data), PSML (energy consumption data), MIMIC (real-world healthcare data from a medical system), and causal_chambers (data describing a physical apparatus).

2. Task Types

T3 focuses on understanding and reasoning questions based on historical information. T4 focuses on forecasting and prediction questions.

3. Option Selection Guidance

Please avoid choosing Uncertain whenever possible. Select Uncertain only when the question is genuinely too ambiguous or difficult to infer from the provided context.

4. External Knowledge Policy

If you encounter unfamiliar domain knowledge (for example, medication effects), you may consult reference materials, including asking GPT for background knowledge. However, do not ask an LLM to directly complete the annotation questions for you.
